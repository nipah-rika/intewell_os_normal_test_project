# Intewell_OS_Normal_Test_Project


## 1.项目以文件夹为单位，命名方式：设备型号+测试组件+测试目的+工程类型，大驼峰方式。

## 2.说明：

    测试E2000d平台，测试TTOS系统中，钩子函数测试。

## 3.编程风格：

         函数linux/类unix风格
         局部变量：匈牙利命名法
         全局变量：匈牙利命名法
         宏定义：linux风格 大写
		 

## 4.程序演示逻辑说明：

此例程演示TTOS的钩子函数测试。

## 5.此项目为TTOS的App应用项目, 需配合对应的CFG（Hypervisor）项目作为测试。

工程完成，创建于2024.01.11，可在IDE中导入打开此工程并编译下载运行。


## 6.测试说明：

### 6.1. 软件环境：

	win10（开发host端）
	MobaXterm开发工具
	开发IDE版本：Intewell_Hyper_I-V2.0.0_UVC_II_C1_Demo
	可连通内网（网段10.10.8.x，或其他网段）


### 6.2. 硬件环境：

	铁科院E2000d开发板
	TLL串口线（调试串口）
	12V电源
	网线若干
	交换机
