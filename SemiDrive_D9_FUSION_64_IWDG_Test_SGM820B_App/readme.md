## 1.项目以文件夹为单位，命名方式：设备型号+测试组件+测试目的+工程类型，大驼峰方式。 
    SemiDrive_D9_FUSION_64_I2c_TestIoExpander_App

## 2.说明：

    试菲森电力SemDrive D9平台，测试TTOS系统中使用外部IWDG驱动。

## 3.编程风格：

         函数linux/类unix风格
         局部变量：匈牙利命名法
         全局变量：匈牙利命名法
         宏定义：linux风格 大写


## 4.程序演示逻辑说明：

此例程演示TTOS的外部看门狗SGM820B的控制方法,GPIO控制,带使能引脚，翻转电平喂狗，超时时间为1.6S左右。

	串口调试工具（putty，secureCRT，powershell等）

	菲森电力 SemiDrive D9主板
