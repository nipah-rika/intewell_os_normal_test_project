此项目为arm的GPOS+RTOS应用项目，还需配合app项目作为测试。
工程完成，创建于2022.07.05，可在IDE中导入打开工程。

测试目的：测试串口驱动的简单使用，包括：打开，控制，发送。让用新户理解如何在TTOS应用中使用串口。

测试说明：
1. 软件环境：
	win10 （开发host端）
	天启萤火虫eca3399c自带ubuntu 18.04系统
	在3399c上18.04系统先安装RTRE for aarch64
	IDE 1.8.2_beta
	可连通内网（网段10.10.8.x）
	
2. 硬件环境：
	win10 PC，开发机
	天启萤火虫eca3399c开发板
	高速usb转串口
	显示器（hdmi接口）
	usb HUB
	usb网卡
	网线
	鼠标键盘
	
3. 详细配合教程/wiki说明：
	http://10.10.8.200/mediawiki/index.php/0x006_Intewell_TTOS_3399串口测试记录
