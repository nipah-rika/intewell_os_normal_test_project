## 1.项目以文件夹为单位，命名方式：设备型号+测试组件+测试目的+工程类型，大驼峰方式。 
    SemiDrive_D9_FUSION_64_Can_Func_Test_App

## 2.说明：

    测试菲森电力SemDrive D9平台，测试TTOS系统的can设备打开、读、写、设置波特率等操作。

## 3.编程风格：

         函数linux/类unix风格
         局部变量：匈牙利命名法
         全局变量：匈牙利命名法
         宏定义：linux风格 大写


## 4.程序演示逻辑说明：

此例程演示TTOS的can设备打开、读、写、设置波特率等操作。

## 5.此项目打开的can设备为CAN0，先发送数据，后接收数据，使用CANPro协议分析平台进行收发

工程完成，创建于2024.03.06，可在IDE中导入打开此工程并编译下载运行。
