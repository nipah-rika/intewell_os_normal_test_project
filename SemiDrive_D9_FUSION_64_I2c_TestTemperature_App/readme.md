## 1.项目以文件夹为单位，命名方式：设备型号+测试组件+测试目的+工程类型，大驼峰方式。 
    SemiDrive_D9_FUSION_64_I2c_TestTemperature_App

## 2.说明：

    试菲森电力SemDrive D9平台，测试TTOS系统中使用I2C获取温度的方法。

## 3.编程风格：

         函数linux/类unix风格
         局部变量：匈牙利命名法
         全局变量：匈牙利命名法
         宏定义：linux风格 大写


## 4.程序演示逻辑说明：

此例程演示TTOS的I2C的控制方法，配置、读写I2C的方法，例程中，循环读取模块温度并打印出结果。
